img的属性

  • src 路径为相对路径 例如src=”./images/jiesong.png”
  • alt 图裂时显示的内容
  • width/height 指定图片的宽/高度
    • 行内样式（一般不这么写）
      • width=”400px”
      • width=”32%”
    • 放在img标签的属性里
      • img {
        width: 400px;
        }
        PS:img的display不太清楚

插入背景图片

  • div {
        background-image: url(./images/jiesong.png);  //设置图片的链接，也是相对地址
    background-size:400px;   //设置图片的大小
    background-repeat: no-repeat;    //设置是否平铺
    width: 400px;    //设置盒子的宽度
    height: 600px;   //设置盒子的高度
}

盒子的宽高属性

  • 不设置高，则无高度
  • 不设置宽度，则默认整个屏幕宽度
  • 父盒子设置了宽高，子盒子继承父盒子的宽度，不继承高度，高度需要自定义。
  • 父盒子只设置了高度，子盒子需自己设置高度，其宽度继承父盒子，即也为整个屏幕宽度

浮动

  • 需要有一个父盒子，占有高度，浮动是飘起来的不占有高度
  • 子盒子紧贴父盒子的内边距对其
  • 添加了浮动后，具有行内块元素的特性（如果盒子不给宽度，由文字的宽度撑开）
  • 父盒子占高度，子盒子都得浮动
  • 但是经常不知道子盒子的高度是多少，所以父盒子的高度往往不写，但是这样子盒子浮动就不占有高度，于是会对其他的盒子造成影响，所以需要清除浮动带来的影响

  • 清除浮动带来的影响的三种方法（配合父盒子无高度的情况使用，若父盒子有高度，则不需要清除浮动带来的影响）

    • 1.在子盒子后面加一个空盒子,并且添加如下内容

    • 2.父级添加overflow方法

    • 3.after伪元素清除浮动(本来应该是 ::after,但是 :after也可以为了兼容)

    • 4.双伪元素清除浮动

块元素，行内元素，行内块元素（ps：还需要补充）

  • 块元素(block) div
  • 行内元素(inline) span 给了高度，宽度也不显示 宽度由span的文字宽度撑开
  • 行内块元素(inline-block) 给了高度和宽度就显示宽高，不给宽度就由文字宽度撑开

li标签 ,a标签

  • li标签是块级元素
  • a标签是行内元素

定位（定位方式+边偏移）

  • 定位方式
    • 静态定位
    • 相对定位
    • 绝对定位
    • 固定定位
  • 边偏移
    • top
    • left
    • right
    • bottom
      —————-这是一条分界线—————-
  • 静态定位
    定位方式不写，默认静态定位，边偏移对它无效，所以静态定位方式一般用来清除定位
  • 相对定位
    • relative 不脱标的标准流
    • 移动位置是相对自己
  • 绝对定位
    • absolute 完全脱标的
      • 如果父盒子没有定位 子盒子移动参考浏览器，而不是自身，并且如果有三个子盒子，盒子2移动，盒子3会填补盒子2的位置，因为绝对定位是脱标的
  • 固定定位
    • 他的位置与父盒子的定位方式无关，只与浏览器有关
    • 固定定位完全脱标，不占有位置，不随滚动条的滚动而滚动
