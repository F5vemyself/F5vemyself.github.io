###数组相关操作

#####数组的定义及使用

  • 方法一：var arr1 = new Array(‘a’, ‘b’, ‘c’); //这是一个预定义的数组，在创建时初始化
  • 方法二： var arr2 = [‘a’, ‘b’, ‘c’ ]; //同样是在创建时初始化，但是这种创建更为简洁直观

  • 方法三 var arr3 = new Array( ); var arr4 = [ ]; //这两种是创建空的数组

  • 可以理解为new Array()=[]

ps：
①方法一、方法二常用 ，其中初期可以多用第一种掌握数组与对象的关键词,后面为了简洁高效用第二种
②踩坑，不要用name作为数组名，会出现错误输出name[0]显示为l，而不是lsc，用Name即可避免问题
